<h2>🛠️ Installation Steps:</h2>

<p>1. Pull Project</p>

```
git clone https://github.com/Mohzaki20/new-event-React.git
```

<p>2. Install Packages</p>

```
npm init
yarn install
```

<p>3. Run Project</p>

```
npm start
```
